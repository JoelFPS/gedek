You need to install the following packages with commands:

pip install python-telegram-bot

pip install pyopt

pip install opencv-python

pip install googlemaps

pip install haversine

pip install geopy

pip install nominatim

pip install db-sqlite3

pip install qrcode

------------------------
Constants.py:
You also need to add your Maps' API and Bot's API to Environment Variables (in Windows)

------------------------

* main.py - calling file
* responses.py - instructions to user messages
* randomizer.py - Geo app (randomization of locations and photos from maps)
* Constants.py - important information for bot

* opt.py - 2fa security

* dbConnector.py - all of operations on the database
